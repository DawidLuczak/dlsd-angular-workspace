# DlsdAngularUi

This is Angular UI components library with custom implementations and without using any other UI libraries.

## Features

- **Buttons**

- **Dropdowns**

- **Forms**:

  - **Inputs**:

    - **Input**

    - **Select**

    - **Timepicker**

    - **Datepicker**

    - **Checkbox**

    - **Radio Button**

    - **Switch**

  - **Controls**:

    - **Form Controls Group**

    - **Form Control Errors**

- **Pagination**

- **Tabs**

- **Toasts**

- **Tooltips**
